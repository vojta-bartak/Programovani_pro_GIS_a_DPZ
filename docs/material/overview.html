Cíl: Napsat a spustit první Python program, pochopit základní datové typy

  • int, float, string, bool
  • Proměnné a přiřazení
  • Základní operace (+, -, *, /, %, //)
  • input() od uživatele
  • f-strings pro formátování
  • Seznamy: vytvoření, indexování, append, len

  • Kalkulačka (načte 2 čísla, vypíše součet)

Praktický výstup:

  • Funkční .py soubor
  • Program s input/output
  • Základní práce se seznamy

Poznámky:

  • Seznamy jen ZÁKLADY (vytvoření, indexování, append, len)
  • Detaily (slicing, metody) až v lekcích 4-5

Lekce 4: Podmínky a cykly I - Fibonacci/Faktoriál

Cíl: Naučit se rozhodování (if) a opakování (for, while), aplikovat na matematické úlohy

Obsah (90 min):

1. Podmínky (25 min)

  • if/elif/else
  • Porovnávací operátory (==, !=, <, >, <=, >=)
  • Logické operátory (and, or, not)
  • Příklady: kategorizace čísel, kontrola věku

2. For cykly (25 min)

  • Iterace přes range()
  • Iterace přes seznam
  • Akumulace výsledků (suma, součin)

3. While cykly (15 min)

  • Cyklus s podmínkou
  • break, continue

4. Praktická úloha (25 min)

  • Společně: n-tý člen Fibonacciho posloupnosti
  • Samostatně: faktoriál čísla

Fibonacci - společně (krok za krokem):

# F(0) = 0, F(1) = 1
# F(n) = F(n-1) + F(n-2)

n = int(input("Který člen? "))

# Iterativní řešení:
a, b = 0, 1
for i in range(n):
    a, b = b, a + b
    
print(f"{n}. člen Fibonacciho posloupnosti: {a}")

Faktoriál - samostatně:

# Studenti zkusí sami
# 5! = 5 * 4 * 3 * 2 * 1

Domácí úkol:

  • Dokončit faktoriál (pokud nestihli)
  • Variace: Fibonacci/faktoriál pro seznam čísel

Poznámky:

  • Fibonacci je klasika - dobře ukazuje logiku cyklů
  • Faktoriál je dobrý samostatný úkol pro procvičení
  • Obojí užitečné pro pochopení akumulace v cyklu

Lekce 5: Podmínky a cykly II - Prvočísla + třídění

Cíl: Prohlubování cyklů, algoritmické myšlení, vytvoření třídícího algoritmu

Obsah (90 min):

1. Rekapitulace (10 min)

  • Dotazy k Fibonacci/Faktoriálu
  • Rychlá kontrola domácích úkolů

2. Prvočísla (30 min)

  • Co je prvočíslo?
  • Algoritmus: zkouška dělitelnosti
  • Společně: “Je číslo N prvočíslo?”
  • Samostatně: “Najdi všechna prvočísla do 100”

3. Třídění - guided discovery (40 min)

  • Zadání: Seřaď seznam [5, 2, 8, 1, 9]
  • Brainstorming: Jak byste to udělali ručně?
  • Studenti většinou navrhnou:
      1. Najít nejmenší, přesunout → Selection sort
      1. Porovnávat sousedy, vyměnit → Bubble sort
  • Společně implementovat OBA algoritmy
  • Vizualizace na tabuli

4. Vnořené cykly (10 min)

  • Ukázat, že třídění = vnořené cykly
  • Propojení s Model Builder: “Pamatujete vnořené modely? V Pythonu je to přirozené!”

Prvočísla - společně:

n = int(input("Číslo: "))
je_prvocislo = True

for i in range(2, n):
    if n % i == 0:
        je_prvocislo = False
        break

if je_prvocislo:
    print(f"{n} je prvočíslo")
else:
    print(f"{n} není prvočíslo")

Bubble sort - společná implementace:

seznam = [5, 2, 8, 1, 9]

for i in range(len(seznam)):
    for j in range(len(seznam) - 1 - i):
        if seznam[j] > seznam[j + 1]:
            # Výměna
            seznam[j], seznam[j + 1] = seznam[j + 1], seznam[j]

print(seznam)

Domácí úkol (alternativy):

Varianta A: - Vymyslet a implementovat vlastní třídící algoritmus

Varianta B: - Najdi všechna prvočísla do 1000 - Nebo: N-té prvočíslo

Poznámky:

  • Guided discovery funguje skvěle - studenti často sami přijdou na algoritmy!
  • Mít připravený backup plán, pokud studenti nevymyslí
  • 40 min na třídění může být málo - případně zkrátit prvočísla
  • DŮLEŽITÉ: Propojit s Model Builder vnořenými modely!

Lekce 6: Funkce a moduly

Cíl: Naučit se psát znovupoužitelný kód, organizovat program do funkcí a modulů

Obsah (90 min):

1. Rekapitulace - proč funkce? (10 min)

  • “Psali jste Fibonacci/faktoriál vícekrát stejně?”
  • DRY principle (Don’t Repeat Yourself)
  • Funkce = pojmenovaný kus kódu, který můžeme volat

2. Definice funkcí (30 min)

  • def, parametry, return
  • Dokumentační stringy (docstrings)
  • Funkce s více parametry
  • Default hodnoty parametrů
  • Rozdíl mezi print() a return

3. Přepis úloh na funkce (30 min)

  • Společně: fibonacci(n)
  • Samostatně:
    • faktorial(n)
    • je_prvocislo(n)
    • bubble_sort(seznam)

4. Moduly (20 min)

  • Vytvoření vlastního modulu (math_utils.py)
  • import math_utils
  • from math_utils import fibonacci
  • Důležité vestavěné moduly: math, os, csv
  • PREVIEW: import arcpy

Příklad - funkce:

def fibonacci(n):
    """Vrátí n-tý člen Fibonacciho posloupnosti.
    
    Args:
        n: Pořadové číslo členu (int)
    
    Returns:
        n-tý člen (int)
    """
    a, b = 0, 1
    for i in range(n):
        a, b = b, a + b
    return a

# Použití:
vysledek = fibonacci(10)
print(f"10. člen: {vysledek}")

Praktický výstup - struktura projektu:

math_utils.py:
--------------
def fibonacci(n):
    """Vrátí n-tý člen Fibonacciho posloupnosti."""
    ...
    
def faktorial(n):
    """Vrátí faktoriál čísla n."""
    ...

def je_prvocislo(n):
    """Zjistí, zda je číslo prvočíslo."""
    ...

def bubble_sort(seznam):
    """Seřadí seznam bublinovým tříděním."""
    ...

main.py:
--------
import math_utils

n = int(input("Zadej číslo: "))
print(f"Fibonacci: {math_utils.fibonacci(n)}")
print(f"Faktoriál: {math_utils.faktorial(n)}")
print(f"Je prvočíslo: {math_utils.je_prvocislo(n)}")

Poznámky:

  • Důležité: Vysvětlit rozdíl mezi print() a return
  • Studenti často nerozumí, proč funkce “nic nedělá” (když vrací hodnotu, ale nevypisuje)
  • Modul = příprava na strukturu ArcPy skriptů
  • Zdůraznit: import arcpy budou používat pořád!

Lekce 7: Textové soubory I - Jednoduchá tabulka

Cíl: Naučit se číst a zapisovat textové soubory, zpracovat jednoduchou tabulku

Obsah (90 min):

1. Čtení souborů (30 min)

  • open(), read(), readlines()
  • with statement (context manager)
  • Zpracování řádků (.strip(), .split())
  • Encoding (utf-8)

2. Zápis souborů (15 min)

  • Režimy: ‘r’, ‘w’, ‘a’
  • Vytvoření nového souboru
  • Zápis výsledků

3. Praktická úloha - jednoduchá tabulka (45 min)

  • Formát: hodnoty oddělené mezerami/taby
  • Data: např. seznam měst s populací
Praha 1300000
Brno 380000
Ostrava 290000
Plzeň 170000
Liberec 103000
  • Úkoly:
    • Načti soubor
    • Vypiš města nad 500000 obyvatel
    • Spočítaj celkovou populaci
    • Najdi město s nejvyšší/nejnižší populací
    • Zapiš výsledky do nového souboru

Příklad:

# Čtení a zpracování
with open('mesta.txt', 'r', encoding='utf-8') as f:
    for radek in f:
        radek = radek.strip()
        if radek:  # neprázdný řádek
            casti = radek.split()
            mesto = casti[0]
            populace = int(casti[1])
            
            if populace > 500000:
                print(f"{mesto}: {populace:,} obyvatel")

# Zápis výsledků
with open('vysledky.txt', 'w', encoding='utf-8') as f:
    f.write("Velká města:\n")
    f.write(f"Praha: 1,300,000\n")

Poznámky:

  • Jednoduchý formát (whitespace separated) - bez komplikací s čárkami
  • Základní operace: čtení, parsování, filtrování, zápis
  • Připravuje na složitější CSV tabulky v příští lekci

Lekce 8: Textové soubory II - CSV tabulka

Cíl: Naučit se pracovat s CSV formátem, zpracovat reálnější data

Obsah (90 min):

1. CSV formát (15 min)

  • Comma-Separated Values
  • Problém: čárky v datech, speciální znaky
  • Proč používat csv modul místo ručního parsování

2. Čtení CSV (25 min)

  • csv.reader() - základní
  • csv.DictReader() - lepší pro data s hlavičkami
  • Práce s hlavičkami
  • Převody datových typů

3. Zápis CSV (10 min)

  • csv.writer()
  • csv.DictWriter() - doporučeno
  • newline='' parametr

4. Praktická úloha - reálnější data (40 min)

  • CSV s daty o okresech:
Okres,Populace,Rozloha,Hustota
Praha,1300000,496,2621
Brno,380000,230,1652
Ostrava,290000,214,1355
Plzeň,170000,261,651
  • Úkoly:
    • Načti CSV pomocí DictReader
    • Filtruj okresy (např. populace > 200000)
    • Spočítej statistiky:
      • Průměrná populace
      • Průměrná hustota
      • Max/min hodnoty
    • Použij funkce z math_utils!
    • Vytvoř nový CSV s výsledky

Příklad:

import csv

# Čtení s DictReader
okresy = []
with open('okresy.csv', 'r', encoding='utf-8') as f:
    reader = csv.DictReader(f)
    for row in reader:
        okresy.append({
            'nazev': row['Okres'],
            'populace': int(row['Populace']),
            'rozloha': float(row['Rozloha']),
            'hustota': float(row['Hustota'])
        })

# Zpracování (filtrace)
velke_okresy = [o for o in okresy if o['populace'] > 200000]

# Statistiky
prumer_pop = sum(o['populace'] for o in okresy) / len(okresy)
print(f"Průměrná populace: {prumer_pop:,.0f}")

# Zápis výsledků
with open('vysledky.csv', 'w', encoding='utf-8', newline='') as f:
    fieldnames = ['nazev', 'populace', 'hustota']
    writer = csv.DictWriter(f, fieldnames=fieldnames)
    
    writer.writeheader()
    writer.writerows(velke_okresy)

Propojení s ArcPy:

  • “Atributové tabulky v ArcGIS vypadají podobně!”
  • “V ArcPy budete exportovat výsledky do CSV”
  • “Nebo načítat CSV a vytvářet z nich feature classes”

Poznámky:

  • csv modul je důležitý - budou ho používat v ArcPy
  • List comprehensions lze lehce ukázat ([x for x in ...])
  • Realistická data připomínají atributové tabulky z GIS!

Lekce 10: Lehký úvod do OOP - Třída Pes

Cíl: Pochopit základní koncept objektového programování, připravit se na práci s ArcPy objekty

Obsah (90 min):

1. Co je objekt? (10 min)

  • Objekt = věc s vlastnostmi (atributy) + schopnostmi (metody)
  • Třída = šablona/recept pro vytváření objektů
  • Instance = konkrétní objekt vytvořený z třídy
  • Analogie: Třída = plánek domu, Instance = konkrétní dům

2. Vytvoření třídy Pes (40 min)

  • Definice třídy (class)
  • Konstruktor __init__
  • Parametr self
  • Atributy (jmeno, vek, rasa)
  • Metody (stekej, oslavPrazdniny)

3. Použití třídy (20 min)

  • Vytváření instancí
  • Volání metod
  • Přístup k atributům
  • Více instancí ze stejné třídy

4. Propojení s ArcPy (20 min)

  • “V ArcPy budete POUŽÍVAT objekty, ne je vytvářet!”
  • Příklady z ArcPy:
    • arcpy.Describe() → objekt s atributy
    • desc.spatialReference.name → objekt.atribut.atribut
    • geometry.area → objekt.atribut
    • geometry.buffer(100) → objekt.metoda()
    • cursor.insertRow() → objekt.metoda()
  • “Teď už rozumíte zápisu objekt.metoda()!”

Příklad - třída Pes:

class Pes:
    """Třída reprezentující psa."""
    
    def __init__(self, jmeno, vek, rasa):
        """Vytvoří nového psa.
        
        Args:
            jmeno: Jméno psa (str)
            vek: Věk v letech (int)
            rasa: Rasa psa (str)
        """
        self.jmeno = jmeno
        self.vek = vek
        self.rasa = rasa
    
    def stekej(self):
        """Pes zaštěká."""
        print(f"{self.jmeno}: Haf haf!")
    
    def oslavPrazdniny(self):
        """Pes oslaví narozeniny."""
        self.vek += 1
        print(f"{self.jmeno} má teď {self.vek} let")
    
    def __str__(self):
        """Textová reprezentace psa."""
        return f"{self.jmeno} ({self.rasa}, {self.vek} let)"

# Použití:
alik = Pes("Alík", 3, "Labrador")
alik.stekej()           # Alík: Haf haf!
print(alik.vek)         # 3
alik.oslavPrazdniny()   # Alík má teď 4 let
print(alik)             # Alík (Labrador, 4 let)

# Další pes ze stejné třídy:
beny = Pes("Beny", 5, "Ovčák")
beny.stekej()           # Beny: Haf haf!

# Dva nezávislé objekty:
print(alik.vek)  # 4
print(beny.vek)  # 5

Cvičení:

Úkol 1: Vytvoř třídu Okres s atributy: - nazev - populace - rozloha

A metodami: - vypocitej_hustotu() - vrátí hustotu - je_velky() - vrátí True pokud populace > 100000

Úkol 2: Vytvoř 3 instance (okresy) a otestuj metody

Propojení s ArcPy - ukázky:

# V ArcPy budete psát (bez vytváření vlastních tříd):

# Describe objekt
desc = arcpy.Describe("vrstva.shp")
print(desc.shapeType)              # objekt.atribut
print(desc.spatialReference.name)  # objekt.atribut.atribut

# Geometry objekt (z kurzoru)
with arcpy.da.SearchCursor("vrstva", ["SHAPE@"]) as cursor:
    for row in cursor:
        geom = row[0]              # Geometrie je objekt!
        print(geom.area)           # objekt.atribut
        print(geom.length)         # objekt.atribut
        buffer = geom.buffer(100)  # objekt.metoda()

Poznámky:

  • DŮLEŽITÉ: Studenti NEMUSÍ psát složité třídy!
  • Cíl: Pochopit, jak POUŽÍVAT objekty z knihoven
  • Consumer, not creator
  • __str__ lze ukázat jako bonus (pro print)
  • Třída Pes je dobrá - jednoduchá, pochopitelná, relatable
  • Na konci VŽDY propojit s ArcPy!

    • Consumer, not creator
    • Geometry, Describe, Cursor objekty

Rizika a jak je řešit:

  1. Studenti budou různě rychlí
    • Řešení: Volitelné bonusové úkoly pro rychlejší
  2. Třídění může být těžké
    • Řešení: Guided discovery, ale mít připravený backup
  3. OOP může být abstraktní
    • Řešení: Jednoduchá třída (Pes), praktické příklady z ArcPy

Co vynechat:

  • Dictionary, tuple (až v ArcPy lekcích)
  • Try/except (až při ArcPy error handling)
  • Lambda funkce (není potřeba)
  • List comprehensions (jen lehce ukázat)
  • Pokročilé OOP (dědičnost, polymorphismus)

  • Lekce 6: Import = jako používání nástrojů z toolboxu
